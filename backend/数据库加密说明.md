# 数据库配置加密说明

## 配置方法

### 1. 添加 Jasypt 加密依赖

已在 `pom.xml` 中添加：

```xml
<dependency>
  <groupId>com.github.ulisesbocchio</groupId>
  <artifactId>jasypt-spring-boot-starter</artifactId>
  <version>3.0.5</version>
</dependency>
```

### 2. 配置文件配置

在 `application.yaml` 中配置加密参数：

```yaml
jasypt:
  encryptor:
    password: your-encryption-password-here
    algorithm: PBEWithMD5AndDES
```

### 3. 使用加密工具

已创建 `JasyptEncryptor.java` 工具类用于加密敏感信息

#### 使用方法：

```bash
# 进入backend目录
cd backend

# 编译项目
mvn clean compile

# 运行加密工具
mvn exec:java -Dexec.mainClass="com.cupk.healthy_diet.util.JasyptEncryptor"

# 或者加密特定值加密单个字符串
mvn exec:java -Dexec.mainClass="com.cupk.healthy_diet.util.JasyptEncryptor" -Dexec.args="要加密的文本"
```

#### 输出示例：

```
原文: your-plain-text
加密: ENC(加密后的字符串)
```

### 4. 更新配置文件

在 `application.yaml` 中使用加密值：

```yaml
spring:
  datasource:
    url: ENC(加密后的URL)
    username: ENC(加密后的用户名)
    password: ENC(加密后的密码)
```

### 5. 安全建议

- 加密密钥请妥善保管，不要泄露！
- 不要将 `application.yaml` 提交到 Git 仓库
- 使用 `application-example.yaml` 作为配置模板
